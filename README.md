# ADHD，目标已落地

一个适合移动端、自用优先的任务与选择辅助网页。所有内容都保存在当前浏览器中，不需要账号，也不会上传任务或选项。

在线使用：<https://wallis1130.github.io/jan-personal/>

## 功能

- **任务连线**：添加 5–36 个任务，自动生成 3×3 至 6×6 棋盘；重复任务同步完成；横、竖或对角连成一线即 BINGO。
- **帮你选择**：按类别管理候选项，随机抽取结果，可确认或再换一个。
- **本地记录**：保存每日任务、BINGO 日期、选择类别和确认结果。
- **移动端适配**：可在手机浏览器中使用，也支持添加浏览器书签。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

默认访问 `http://localhost:3000`。

## 发布前检查

```bash
npm run lint
npm test
```

`npm test` 会构建正式版本，并检查任务连线和 Final Call 两个页面能否正常返回。

## 数据说明

数据使用浏览器 `localStorage` 保存，因此：

- 不同设备和不同浏览器之间不会自动同步。
- 清除浏览器网站数据会同时删除任务、记录和选择项。
- 网页本身不收集账号或个人信息。

## 技术结构

项目使用 React、vinext、Vite 和 Cloudflare Worker 兼容输出构建。
